# Teslo Api 


1. Clonar proyecto
```
npm install
```
2. Clonar ```.env.template``` y remombrarlo a ```.env ```
3. Levantar BD con docker
```
docker-compose up -d
```
4. Correr el proyecto
```
npm run start:dev
```
5. Ejecutar seed para llenar datos en BD
```
http://localhost:3000/api/seed
```
